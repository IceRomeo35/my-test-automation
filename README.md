# Test Automation Framework

A beginner-friendly API test automation framework built with Python and pytest.

## What It Does

This framework automates API testing against a public demo API (JSONPlaceholder). It demonstrates:
- Reading test data from JSON files
- Making HTTP requests (GET, POST)
- Validating response status codes and data
- Generating HTML test reports

## Tech Stack

- Python 3.x
- pytest (test framework)
- requests (HTTP library)
- pytest-html (report generation)
